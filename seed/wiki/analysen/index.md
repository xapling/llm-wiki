# Analysen

Wenn eine Anfrage an das Wiki eine übertragbare Erkenntnis produziert (ein
Muster, eine Synthese, eine Bewertung), wird sie hier als eigene Seite
gespeichert — die Exploration selbst wird zum dauerhaften Artefakt.

*Noch keine gespeicherten Analysen.*
